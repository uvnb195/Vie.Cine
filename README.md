## Features 

- Tra cứu thông tin phim/ diễn viên
- Tra cứu suất chiếu
- Đặt hẹn thông báo phim ra mắt
- Đặt vé xem phim
- Thanh toán trực tuyến
- Tích hợp Dark Mode
